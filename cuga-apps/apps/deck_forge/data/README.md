# DeckForge Sample Data

Drop your source files here and DeckForge will turn them into a presentation.

## Supported formats

| Format | Extensions |
|--------|-----------|
| PDF documents | `.pdf` |
| PowerPoint slides | `.pptx`, `.ppt` |
| Markdown / plain text | `.md`, `.markdown`, `.txt`, `.rst` |
| Audio recordings | `.mp3`, `.wav`, `.m4a` |
| Video recordings | `.mp4`, `.webm`, `.mov`, `.avi` |

Files in any other format are silently ignored by the agent.

## Usage

1. Copy your source files into this directory (or a subdirectory).
2. In the DeckForge UI, enter the path to this directory.
3. Enter your presentation topic and click **Generate Deck**.
